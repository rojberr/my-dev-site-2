JayBeeHoney Webshop Project

  • Category: eCommerce, Spring Boot, Web design, Gradle, Angular
  • Client: Me, myself and I
  • Project date: 01 March, 2021
  • Project URL: jayhoney.herokuapp.com/

Wanna buy delicious honey? Theres no better place to do that! üçØ

As my family owns a bee-garden, I decided to create a webshop where we could sell our vast ammount of honey, that we produce each year. I decided to use:

  • Java ( currently 17 LTS),
  • Spring Boot with JPA and Hibernate
  • Build tools ( Gradle, Maven ),
  • Selenium and JUnit 4 / 5 for Unit- and E2E testing,
  • Docker (Dockerfile, docker-compose),
  • Jenkins,
  • ... and a little bit of Javascript with Angular.
This project waits to be launched, because we need to do the paperwork related to hygiene and food tests (this is how burocratics slow down the process of development üòè). To look at the code or to get even more information go to:
  • https://github.com/rojberr/honey - for Angular frontend code,
  • https://github.com/rojberr/honey - for Spring Boot backend code,
