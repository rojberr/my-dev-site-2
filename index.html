Projects

My projects were created to learn technologies, to help others but mostly to also have some fun. They were created alone and in team. They include: a Speedysalesman - API solving the traveling salesman problem, CertGen - Spring Boot BouncyCastle API for certificates creation, Gradle Jenkins CI/CD builder, OpenGL 3D Chess replayer, Arduino plants waterer robot and data-analytics Docker metabase dashboard.
