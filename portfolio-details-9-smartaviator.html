Project information

  • Category: Rest API, Spring Boot
  • Client: Me, myself and I
  • Project date: 05 March, 2022
  • Project URL: github.com/rojberr/smartaviator

Did you know that aviation applications need to deal with 24 different time zones?

This application demonstrates how to precisely implement time in Java env. It is able to save, retrieve and operate with time and be precise up to 1 milisecond. This application includes Spring Boot API with 2 REST access points /cargo and /flight (plus /init for test purposes) and PostgreSQL database initialized with docker-compose. Exact usage is explained in the README.md file. It was fun playing with Java time datatypes, and it will be surely helpful in many more different projects. To look at the code or to get even more information go to:
